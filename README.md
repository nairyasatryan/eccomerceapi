# Backend for e-commerce web site

## Description

This project waas built for e-commerce application for adding data, products, users, etc.



## Features

- User registration and login
- Authetication with JWT
- Simple Crud
- Admin verification
- Payment with stripe
- Security of user's cart data
- MongoDB database
